Sympy常用函数总结

基础

from sympy import *

数学格式输出：

xxxxxxxxxx
init_printing()

添加变量：

xxxxxxxxxx
x, y, z, a, b, c = symbols('x y z a b c')

声明分数：

xxxxxxxxxx
Rational(1, 3)

 

 

化简式子：

xxxxxxxxxx
simplify((x**3 + x**2 - x - 1)/(x**2 + 2*x + 1))

 

 

因式分解：

xxxxxxxxxx
expand((x + 2)*(x - 3))

 

 

提取公因式：

xxxxxxxxxx
factor(x**3 - x**2 + x - 1)

 

 

约分：

xxxxxxxxxx
cancel((x**2 + 2*x + 1)/(x**2 + x))

 

 

裂项：

xxxxxxxxxx
apart((4*x**3 + 21*x**2 + 10*x + 12)/(x**4 + 5*x**3 + 5*x**2 + 4*x))

 

 

变换形式：

xxxxxxxxxx
tan(x).rewrite(sin)

 

 

数列求和：

xxxxxxxxxx
Sum(x ** 2, (x, 1, a)).doit()

 

 

数列求积：

xxxxxxxxxx
Product(x**2,(x, 1, a)).doit()

 

 

微积分

求导：

xxxxxxxxxx
diff(cos(x), x)

 

 

求高阶导：

xxxxxxxxxx
diff(x**4, x, 3)

 

 

连续求偏导：

xxxxxxxxxx
diff(exp(x*y*z), x, y, 2, z, 4)

 

 

不定积分：

xxxxxxxxxx
integrate(cos(x), x)

 

 

定积分:

xxxxxxxxxx
integrate(exp(-x), (x, 0, oo))

 

 

多重积分：

xxxxxxxxxx
integrate(exp(-x**2 - y**2), (x, -oo, oo), (y, -oo, oo))

 

 

极限：

xxxxxxxxxx
limit(sin(x)/x, x, 0)

 

 

泰勒展开（到第4阶）：

xxxxxxxxxx
sin(x).series(x, 0, 4)

 

 

泰勒展开（在x=6处）：

xxxxxxxxxx
exp(x - 6).series(x, 6)

 

 

矩阵

矩阵求逆：

xxxxxxxxxx
Matrix([[1, 3], [-2, 3]])**-1

 

 

求转置：

xxxxxxxxxx
Matrix([[1, 2, 3], [4, 5, 6]]).T

 

 

生成单位矩阵：

xxxxxxxxxx
eye(3)

 

 

求行列式：

xxxxxxxxxx
Matrix([[1, 0, 1], [2, -1, 3], [4, 3, 2]]).det()

 

 

化成行阶梯形矩阵：

xxxxxxxxxx
Matrix([[1, 0, 1, 3], [2, 3, 4, 7], [-1, -3, -3, -4]]).rref()

 

 

求列向量空间：

xxxxxxxxxx
Matrix([[1, 1, 2], [2 ,1 , 3], [3 , 1, 4]]).columnspace()

 

 

xxxxxxxxxx
M = Matrix([[3, -2,  4, -2], [5,  3, -3, -2], [5, -2,  2, -2], [5, -2, -3,  3]])

求特征值：

xxxxxxxxxx
M.eigenvals()

 

 

求特征向量：

xxxxxxxxxx
M.eigenvects()

 

 

求对角化矩阵，返回两个矩阵P、D满足：

xxxxxxxxxx
M.diagonalize()

 

 

解方程

求解集：

xxxxxxxxxx
solveset(x**2 - x, x)

 

 

求解集（显示多少个重根）：

xxxxxxxxxx
roots(x**3 - 6*x**2 + 9*x, x)

 

 

求解集（用Eq构造等式）：

xxxxxxxxxx
solveset(Eq(sin(x), 1), x, domain=S.Reals)

 

 

解线性方程组：

xxxxxxxxxx
linsolve([x + y + z - 1, x + y + 2*z - 3 ], (x, y, z))

 

 

解线性方程组（矩阵表示）：

xxxxxxxxxx
linsolve(Matrix(([1, 1, 1, 1], [1, 1, 2, 3])), (x, y, z))

 

 

解非线性方程组：

xxxxxxxxxx
nonlinsolve([exp(x) - sin(y), 1/y - 3], [x, y])

 

 

解微分方程：

xxxxxxxxxx
f, g = symbols('f g', cls=Function)
xxxxxxxxxx
dsolve(Eq(f(x).diff(x, x) - 2*f(x).diff(x) + f(x), sin(x)), f(x))

 

 

解不等式组：

xxxxxxxxxx
from sympy.solvers.inequalities import *
xxxxxxxxxx
reduce_inequalities([x <= x ** 2], [x])

 

 

逻辑代数

xxxxxxxxxx
from sympy.logic.boolalg import is_cnf

合取范式：

xxxxxxxxxx
to_cnf(~(x | y) | z)

 

 

析取范式：

xxxxxxxxxx
to_dnf(x & (y | z))

 

 

化简逻辑函数：

xxxxxxxxxx
simplify_logic((~x & ~y & ~z) | ( ~x & ~y & z))

 

 

xxxxxxxxxx
from sympy.logic import *

化简最小项之和为析取范式

xxxxxxxxxx
minterms = [0, 7]
SOPform([x, y, z], minterms)

 

 

化简最小项之和为合取范式

xxxxxxxxxx
minterms = [[1, 0, 1], [1, 1, 0], [1, 1, 1]]
POSform([x, y, z], minterms)

 

 

化简最小项之和为析取范式（第7项任取）

xxxxxxxxxx
minterms = [[1, 0, 1], [1, 1, 0]]
dontcares = [7]
SOPform([x, y, z], minterms, dontcares)

中国剩余定理解同余方程（模数需互质，前三个数为模数，后三个数为余数，返回第一个数为结果）：

xxxxxxxxxx
crt([99, 97, 95], [49, 76, 65])

 

 

解同余方程（模数不需互质但比中国剩余定理慢，每个元组第一个数为余数，第二个数为模数，返回第一个数为结果）：

xxxxxxxxxx
